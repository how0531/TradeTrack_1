
# TradeTrack Pro V1.3.1 使用者操作指南

歡迎使用 **TradeTrack Pro**。這是一款專為追求穩定獲利的交易者打造的績效管理系統。本指南將協助您掌握從基礎記錄到高階策略分析的所有功能。

---

## 1. 快速入門 (Getting Started)

### 帳戶與同步
*   **離線模式 (預設)**：初次使用時，所有資料僅儲存於您的瀏覽器中。此模式下無需網路即可操作，且保有極高的隱私性。
*   **雲端同步 (推薦)**：
    1.  進入 **設定 (Settings)** > **Google 雲端備份**。
    2.  點擊登入後，系統會自動將本地資料加密上傳至雲端。
    3.  **跨裝置使用**：在手機或電腦登入同一帳號，交易記錄將即時同步。
*   **資料衝突處理**：若系統偵測到本地與雲端資料不一致（例如在離線時新增了交易），會彈出視窗詢問您要「合併」或「捨棄」本地資料，確保數據不遺失。

---

## 2. 介面導覽 (Dashboard)

應用程式底部設有懸浮導航列，分為四大核心模組：

1.  **總覽 (Stats)**：您的戰情室，監控資金曲線、回撤與策略表現。
2.  **日誌 (Journal)**：以月曆形式檢視每日損益分佈。
3.  **新增 (+)**：中央懸浮按鈕，快速記錄交易。
4.  **記錄 (Logs)**：詳細的交易歷史清單（時間軸模式）。
5.  **設定 (Settings)**：帳戶管理、風險控管與備份。

---

## 3. 功能詳細說明

### A. 總覽 (Stats) - 資金與策略分析
*   **即時資訊列**：
    *   **隱私模式**：點擊右上角「眼睛」圖示可隱藏金額（顯示為 ****），適合在公共場合檢視。
    *   **帳戶切換**：左上角可切換顯示「全部帳戶」或「特定帳戶」的績效。
    *   **篩選器 (Filter)**：點擊漏斗圖示，可針對特定「策略」或「心態」進行交叉分析。
*   **權益曲線圖 (Equity Curve)**：
    *   **互動式調整**：按住圖表下方的橫條 (Drag Handle) 可上下拖曳，調整圖表高度。
    *   **視覺標記**：藍線為資金曲線，黃點代表創歷史新高 (New Peak)。
    *   **雙重資訊**：背景隱約顯示的回撤圖 (Drawdown Bar)，讓您同時掌握獲利與風險。
*   **策略績效矩陣**：
    *   切換右上角的圖示可選擇檢視模式：
        *   **列表模式**：詳細列出各策略的勝率、損益與賺賠比。
        *   **氣泡圖模式**：X軸為勝率，Y軸為賺賠比，氣泡大小代表交易次數。位於右上角的策略表現最佳（高勝率、高賺賠比）。

### B. 日誌 (Journal) - 交易月曆
建立交易的一致性與紀律：
*   **年份/月份選擇器**：點擊上方的年份月份（例如 `2024 • 10`），可開啟下拉選單快速跳轉至特定月份。
*   **損益熱力圖**：
    *   **圓點呈現**：實心圓點代表當日有交易。
    *   **顏色區分**：紅色 (Rose) 代表獲利，綠色 (Emerald) 代表虧損。顏色越深代表金額越大。
*   **快速新增**：直接點擊月曆上的日期，即可開啟該日期的交易新增視窗。
*   **月度總結**：底部顯示該月份的總損益、勝率與總交易筆數。

### C. 記錄 (Logs) - 交易流水帳
*   **時間軸檢視**：交易依日期分組排列，左側顯示該日的當日總損益。
*   **詳細資訊**：點擊單筆交易卡片可展開，查看筆記並進行「編輯」或「刪除」。
*   **標籤顯示**：每筆交易皆會顯示所屬的策略與心態標籤，以及歸屬的帳戶顏色。

### D. 新增交易 (Add Trade)
點擊中央的 **「+」** 按鈕：
1.  **歸屬帳戶**：選擇此筆交易屬於哪個帳戶（如：美股、台指期）。
2.  **類型**：點擊切換「獲利 (Profit)」或「虧損 (Loss)」。
3.  **金額**：輸入損益金額（自動計算正負號）。
4.  **策略/心態**：從預設清單中點選，或直接輸入文字建立新標籤。
5.  **筆記**：記錄進出場理由或檢討。

## 4. 設定與進階功能 (Settings)

### 風險管理 (Risk Management)
這是專業交易者的核心功能：
*   **最大回撤閥值 (MDD Threshold)**：設定百分比（例如 20%）。當帳戶回撤觸及此線時，系統會發出警告。
*   **連續虧損次數 (Max Loss Streak)**：設定容忍的連敗次數（例如 3 次）。當連續虧損達到此設定時，系統會在頂部顯示紅色警戒條，強烈建議暫停交易。

### 帳戶管理 (Manage Portfolios)
*   **多帳戶支持**：您可以新增多個帳戶（例如：Main Account, Scalping, Swing）。
*   **顏色自訂**：點擊色環可為每個帳戶設定專屬的代表色，這將反映在圖表與日誌中。
*   **資金設定**：設定各帳戶的初始資金，以便正確計算回報率。

### 數據管理 (Data Management)
*   **備份與還原 (JSON)**：
    *   **匯出 JSON**：將包含設定、帳戶、交易記錄的完整資料庫備份下載。
    *   **匯入 JSON**：從備份檔還原資料（支援版本遷移）。
*   **CSV 整合**：
    *   **匯出 CSV**：將交易記錄匯出為 Excel 可讀格式。
    *   **匯入 CSV**：支援批量匯入歷史交易（請參考匯出的格式進行編輯）。
*   **危險區域**：重置所有資料將清空本地與雲端的所有紀錄，請謹慎使用。

## 5. 安裝與 PWA

**TradeTrack Pro** 支援 PWA (Progressive Web App) 技術：
*   **iOS / Android**：在瀏覽器 (Safari/Chrome) 選單中點擊「加入主畫面 (Add to Home Screen)」。
*   **桌面版**：在 Chrome 網址列右側點擊安裝圖示。
*   **優勢**：安裝後可全螢幕執行，體驗如原生 App，並支援離線操作。
